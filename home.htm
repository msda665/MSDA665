MSDA 665. Big Data Analytics for Business

Course Description

In-depth study of the concepts, methods, and tools for Data Science and Big Data Analytics with a focus on business scenarios. This course is designed to introduce students to advanced analytics techniques that can be applied to big data sets. Topics include python programming, principal component analysis (PCA), data imputation, web scraping, deep learning, natural language processing (NLP), and mathematical optimization. For most techniques, students learn how to apply the techniques to solve real-world business problems. The course covers various analytics tools such as Excel, Python with Google Colaboratory, scikit-learn, BeautifulSoup, Keras, TensorFlow, OpenSolver, and Gurobi Optimizer.
